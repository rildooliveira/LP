### 3. Implementação Técnica (JavaScript)

No arquivo `script.js`, vamos implementar a lógica de cálculo dividida em três tarefas fundamentais:

**Tarefa A: Definição de Variáveis de Produção**
Vamos definir a capacidade máxima do Hub e o volume atual de projetos em desenvolvimento.

```javascript
// Dados de Operação (Inputs)
let capacidadeProjetos = 50; 
let projetosAtivos = 38;

```

**Tarefa B: Cálculo de Eficiência (Operadores Aritméticos)**
Utilizaremos os operadores de divisão (`/`) e multiplicação (`*`) para encontrar a porcentagem de ocupação do Hub.

```javascript
// A lógica matemática (Processamento)
let calculoEficiencia = (projetosAtivos / capacidadeProjetos) * 100;

```

**Tarefa C: Injeção Visual e Atualização de KPIs**
Agora, conectamos o resultado matemático à largura (`width`) da barra de progresso e ao texto da interface.

```javascript
const carregarPerformance = () => {
    document.getElementById("val-ativos").innerText = projetosAtivos;
    document.getElementById("val-meta").innerText = capacidadeProjetos;
    document.getElementById("label-eficiencia").innerText = `Eficiência: ${calculoEficiencia.toFixed(1)}%`;

    // Atualiza a largura da barra dinamicamente
    const barra = document.getElementById("progress-bar-fill");
    barra.style.width = calculoEficiencia + "%";
};

```

## Código Completo: Aula 39 (Evolutivo)

**script.js**

```javascript
/**
 * PROJETO HUB DE INOVAÇÃO - FASE 4
 * Aula 39: Estruturas Condicionais (if/else)
 */

// 1. DADOS DE ENTRADA (Fases Anteriores)
const nomeHub = "Hub de Inovação Tecnológica";
let capacidadeProjetos = 50;
let projetosAtivos = 46; // Mude para 20, 38 ou 46 para testar os alertas

// 2. FUNÇÃO DE LOG (Reutilizável)
const addLog = (msg) => {
    const terminal = document.getElementById("terminal-auditoria");
    const agora = new Date().toLocaleTimeString();
    terminal.innerHTML += `<p>> ${agora}: ${msg}</p>`;
};

// 3. MOTOR DE INTELIGÊNCIA DO HUB
const processarHub = () => {
    // A. Cálculos de Performance (Aula 38)
    let eficiencia = (projetosAtivos / capacidadeProjetos) * 100;

    // B. Mapeamento de Elementos (Aula 36)
    const elLed = document.getElementById("led-status");
    const elBarra = document.getElementById("progress-bar-fill");
    const elStatusBox = document.getElementById("status-box");
    const elStatusMsg = document.getElementById("security-msg");
    const elInstrucao = document.getElementById("security-instruction");

    // C. LÓGICA DE DECISÃO (Tarefa A e B - Aula 39)
    if (eficiencia >= 90) {
        elStatusBox.className = "security-critical";
        elStatusMsg.innerText = "CRÍTICO: SOBRECARGA DE ATIVOS";
        elInstrucao.innerText = "Bloqueio de novos projetos ativado imediatamente.";
        elLed.style.background = "#f87171"; // LED Vermelho
        addLog("ALERTA: Protocolo de segurança nível 3 ativado.");
    } 
    else if (eficiencia >= 70) {
        elStatusBox.className = "security-warning";
        elStatusMsg.innerText = "ATENÇÃO: ALTA OCUPAÇÃO";
        elInstrucao.innerText = "Recomenda-se triagem de novos registros.";
        elLed.style.background = "#fbbf24"; // LED Amarelo
        addLog("AVISO: Sistema operando próximo ao limite operacional.");
    } 
    else {
        elStatusBox.className = "security-normal";
        elStatusMsg.innerText = "SISTEMA ESTÁVEL";
        elInstrucao.innerText = "Hub operando dentro da margem de segurança.";
        elLed.style.background = "#10b981"; // LED Verde (Padrão)
    }

    // D. ATUALIZAÇÃO DA INTERFACE (Sincronização Final)
    document.getElementById("hub-nome").innerText = nomeHub;
    document.getElementById("val-ativos").innerText = projetosAtivos;
    document.getElementById("val-meta").innerText = capacidadeProjetos;
    document.getElementById("label-eficiencia").innerText = `Eficiência: ${eficiencia.toFixed(1)}%`;
    elBarra.style.width = `${eficiencia}%`;

    addLog(`Auditoria de performance concluída (${eficiencia.toFixed(1)}%).`);
};

// Inicializar processamento
processarHub();

```

No arquivo `script.js`, realizaremos a integração final dividida em três tarefas de controle estético:

**Tarefa A: Injeção de Estrutura Enriquecida (innerHTML)**

```javascript
const injetarAlertaHTML = () => {
```

**Tarefa B: Manipulação de Propriedades Diretas (Style Property)**

```javascript
const mudarCorDireta = () => {

```

**Tarefa C: Gestão de Estado Global (classList e Temas)**

```javascript
const alternarTema = () => {

        addLog("SISTEMA: Modo Escuro ativado para economia de energia.");
    }
```
